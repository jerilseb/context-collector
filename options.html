Keyboard Shortcut

To change the keyboard shortcut for this extension:

  1. Go to chrome://extensions/shortcuts in your browser
  2. Find "Context Collector" in the list
  3. Click the input field next to "Collect content from current page (when active)"
  4. Press your desired key combination
  5. Click anywhere outside the input to save

Note: Due to Chrome extension limitations, keyboard shortcuts can only be changed through Chrome's extension settings page.

About

Context Collector helps you quickly extract specific sections of a webpage and convert them into Markdown format.
